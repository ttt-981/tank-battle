# 🎮 坦克大战 (Tank Battle)

一个用纯 HTML/CSS/JavaScript 实现的经典坦克大战游戏，无需任何依赖，直接在浏览器中运行。

## 🎯 玩法

- **↑↓←→** 或 **WASD** —— 控制玩家坦克移动
- **空格键** —— 射击
- **P** —— 暂停/继续
- **R** —— 重新开始

## ✨ 特性

- 🎨 Canvas 渲染，流畅动画
- 🧱 砖块墙壁系统，子弹可摧毁墙壁
- 🤖 AI 敌人，会追踪玩家并射击
- 💥 爆炸粒子特效
- 🔊 Web Audio API 音效
- 📱 移动端触控支持
- 🏆 多关卡，难度递增
- ❤️ 生命系统和得分系统

## 🚀 快速开始

1. 克隆仓库：
   ```bash
   git clone https://github.com/ttt-981/tank-battle.git
   ```

2. 直接用浏览器打开 `index.html` 即可开始游戏！
